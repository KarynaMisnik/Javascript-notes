Code Analysis and Explanation:

fetch("numbers.txt"); is a built-in JS function(Web API). It sends a request lo load a file/URL and returns Promise

.then(res => res.text())

means:"when fetch is done, do next step"
res => is an arrow function, it means the same as function(res){ return res.text(); } res.text() is not yet the text, it is response object. It contains

  • status(200 OK)
  • headers
  • body(file content)
res.text() text() is a built-in JS function of the response object. It reads the body as text. And returns another Promise.
